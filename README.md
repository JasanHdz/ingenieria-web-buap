## ¿Comó funciona?

Requiere Node.JS 

npm install para instalar las dependencias.
npm run dev para el entorno de desarrollo.
npm run build && npm run start para el entorno de producción.

## Tecnologías
  - Next.js
  - Styled-components
  - React.js
